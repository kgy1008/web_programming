Backend

  • Node.js, Express.js, MongoDB

    • 로그인 기능 구현
      • 패스워드와 같은 중요 정보는 Hash 함수를 이용하여 암호화하여 DB에 저장하도록 구현
    • 이미지 파일 저장
      • base64로 인코딩된 이미지를 Buffer.from()을 사용하여 Buffer로 변환하고 이를 파일로 저장하는 방식으로 구현
        • 그 후, 해당 파일의 경로를 생성하고 해당 파일을 URL로 변환하여 DB에 저장하였음
    • 자동 요약 서비스 구현 시도
      • 사용자가 프로젝트 설명란에 아무것도 입력하지 않았을 경우, README.md 파일을 불러와 자동으로 요약을 제공하려 함
      • githubAPI를 이용하여 github 링크의 README.md 파일을 불러오는 방식으로 구현하려고 시도함
      • Naver Clova API를 이용한 내용 자동 요약 구현 시도
        • 그러나 Clova API의 한계로 인해 Readme.md의 문맥을 정확히 파악하지 못하는 문제가 발생
        • 이 문제에 대한 해결책을 모색 중임
