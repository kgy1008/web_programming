Backend

Technologies: Node.js, Express.js, MongoDB

로그인 기능

패스워드와 같은 중요 정보는 Hash 함수를 이용하여 암호화하여 DB에 저장

이미지 파일 저장

base64로 인코딩된 이미지를 Buffer.from()을 사용하여 Buffer로 변환하고 파일로 저장. 이후 파일 경로를 생성하여 URL로 변환하여 DB에 저장

자동 요약 서비스 시도

githubAPI를 이용하여 github 링크의 README.md 파일을 불러오는 방식으로 구현하려고 시도

Naver Clova API를 이용한 내용 자동 요약 구현 시도 중, Readme.md의 문맥을 정확히 파악하지 못하는 문제 발생
